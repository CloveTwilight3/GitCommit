## Installation

### Windows
Install Chocolatey following the steps [here](https://chocolatey.org/install)

Run the command:
```ps
choco install gitcommit
```

### Ubuntu
Install from PPA using:
```sh
sudo add-apt-repository ppa:clovetwilight3/gitcommit
sudo apt update
sudo apt install gitcommit
```

If you donâ€™t already have add-apt-repository:
```sh
sudo apt install software-properties-common
```

### Arch
Add AUR following the steps [here](https://wiki.archlinux.org/title/Arch_User_Repository)


Install from yay using
```sh
yay -S gitcommit
```
